# Wi-Fi Password Security Auditor

A simple Python tool to evaluate the strength of a Wi-Fi password that you already know. This project is intended for security awareness and defensive use only.

## Features

- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Warns about common weak passwords
- Provides a strength rating and improvement suggestions

## Requirements

- Python 3.8 or later

## Installation

```bash
git clone https://github.com/yourusername/wifi-password-auditor.git
cd wifi-password-auditor
```

## Usage

```bash
python wifi_auditor.py
```

Enter your Wi-Fi password when prompted. The tool will display its strength and suggestions for improvement.

## Example

```
Enter your Wi-Fi password:
MyHome@2026

Strength: Very Strong
Excellent! Your password follows good security practices.
```

## Disclaimer

This project **does not crack, recover, or bypass Wi-Fi passwords**. It only evaluates the strength of a password that the user already knows. Use it only for educational and defensive security purposes.
